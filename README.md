# SwasthyaScan

Smart AI-Powered Tuberculosis Detection for Early and Reliable Diagnosis.

## Tech Stack

- React (Vite)
- Tailwind CSS
- React Router
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Included Frontend Modules

- Branded home page with hero, pipeline preview, features, stats, and chatbot
- Login/Register pages with healthcare-focused UI
- Dashboard page with mock operational cards
- Prediction page with interactive animated ML pipeline simulation
- Result page with mock prediction summary, confidence, risk, precautions, suggestions, videos, and fake heatmap
- Dark/light mode toggle and toast notifications

## Note

This is a frontend-only prototype using mock data and simulated ML responses.
